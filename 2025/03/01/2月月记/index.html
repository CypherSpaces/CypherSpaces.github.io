2.8

上午帮妈妈墩地，中午的时候听爸爸说他们单位找了几个人用 deepseek 的模型搭了个自己的 AI，还没有喂数据，想让我下午过去学学怎么搭的。

我下午本来也打算从那边的房子里拿双鞋回来，所以就答应了。

那么其实搭这个好像也没什么太复杂的，先去 ollama 上下载环境和模型，然后配置几个环境变量，再把下载好的模型运行起来就 ok 了。

现在只能在 cmd 上跟 AI 对话，如果想部署到 UI 上的话，可以下个 docker 改改设置就可以自动部署到 UI 上了。

2.5

中午吃饭吃到了闻名已久的折耳根，也称鱼腥草。

做之前妈妈让我闻了闻，味道不太好描述，感觉有一种莫名的清凉感，也没什么特别的腥味。

妈妈把折耳根分成了两碗，一碗是辣椒配料，另一晚像是醋加酱油，因为我不太能吃辣的。

我夹起一根短短细细的折耳根，放到嘴里品尝了一下，的确是有点腥，不是很好吃，倒也没有传闻里难吃的那么夸张，少吃一点还是可以接受的。

后面感觉还是放辣椒它的腥味可以淡一点，所以在加醋的那碗里也放了辣椒。

2.4

今天在网易云上听音乐，每日推荐里听到了羽肿的《Windy Hill》，莫名伤感。去知乎上搜了一下羽肿，想了解了解，结果看到的都是 18 年以前的贴子，感觉知乎上聊的人不是很多。

但是看到了好多 18 年前活跃的账号，从 18 年至今都没有任何活动，突然觉得更伤感了……

每次看到这样很早的账号多年没有活动都会觉得莫名伤感，就像是一个鲜活的灵魂逐渐消失在了时间之中，多年前或许还是年轻朝气，意气风发，现在则是经历了社会的打磨，眼睛里早已不再闪烁当年那样的光，空余麻木的身躯为生存奔波，不再有时间去刷知乎，和知友们分享自己的喜怒哀乐……

然后发现了一个很有意思的事情。我无意中点进了网易 CEO 丁磊的主页里，发现他竟然有 99999 个粉丝，我寻思这么巧吗，就点了个关注，粉丝数从 99999 变成了 100000。

过了两分钟，又看了一眼，发现他怎么又变成 99999 个粉丝了？原来搁着后台改数据呢，该说不说，我确实上钩了，点了个关注 :)
